1. download and install 'Bitmap Font Generator' from 'http://www.angelcode.com/products/bmfont/' as recommended by 
2. in Options>Font Settings, select desired font, size and select 'Render from TrueType outline' press ok
3. in Options>Export Options, add Padding if you need to apply an effect that extends character borders. Select '32' Bit depth for alpha backround and choose the correct Width and Height (trial and error). Select 'XML' Font descriptor. Select png Texture. press ok
4. Options>Save bitmap font as... to export the font .png and atlas .xml
5. (Optional) import resulting .png into PhotoShop to add layer effects inline with designs.
